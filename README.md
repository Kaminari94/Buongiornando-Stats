
# Buongiornando-Stats
![Alpine.js](https://img.shields.io/badge/alpinejs-white.svg?style=for-the-badge&logo=alpinedotjs&logoColor=%238BC0D0)
![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white)
![Apex](https://img.shields.io/badge/Apex-blue?style=for-the-badge&logo=salesforce&logoColor=white)

Piccola dashboard per visualizzare e confrontare i tempi del “buongiornando” (meme interno del nostro gruppo Telegram), con classifiche e grafici.

Il progetto nasce come esercizio pratico per:

- raccogliere dati giornalieri (timestamp del messaggio),
- trasformarli in metriche semplici (“tempi sul giro”),
- visualizzarli in modo chiaro tramite una dashboard web leggera.

**Tech stack:** Alpine.js + TailwindCSS + ApexCharts.

[Visualizza qui la dashboard.](https://kaminari94.github.io/Buongiornando-Stats/)

# Background

Nel nostro gruppo Telegram il “buongiornando” è diventato un rito quotidiano: ho automatizzato la risposta con un bot e, col tempo, ho iniziato a registrare gli orari per creare una classifica e vedere l’andamento nel tempo. Perchè? Perchè no!
